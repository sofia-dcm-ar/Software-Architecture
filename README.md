# Software-Architecture
Final project of "Programming Methodology I", Informatic Engineering course.
Different Software Architectures for POO applied in C# using SharpDevelop version 5.1.0, migrated to Visual Studio 2022

##Course and project instructions:
Acquire knowledge of a new software architecture every week.
Implement each architecture in the existing software by following specific instructions and weekly exercises, with full creative and structural freedom.
Submit weekly assignments focusing exclusively on the previous and new classes required to fulfill the objectives.
Maintain a weekly folder structure to ensure organization, clarity, and easy access to previous versions for tracking modifications.

##Project Organization
Class 1: Experience with interface design and implementation
Class 2: Strategy pattern ; Iterator pattern
Class 3: Factory Method pattern ; Observer pattern
Class 4: Adapter pattern ; Decorator pattern
Class 5: Proxy pattern ; Command pattern
Class 6: Composite pattern ; Template Method pattern
Class 7: Chain of Responsability pattern ; Singletone pattern
